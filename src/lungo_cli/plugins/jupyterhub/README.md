# JupyterHub (plugin for Lungo)

## Configuration

```yaml
# ========
# modules:
# ========

#   # Settings for the JupyterHub module
#   jupyterhub:

#     # Enable or disable the module
#     # Type: boolean
#     # Default: true
#     # Required: no
#     enabled: true

#     # Password for users inside the container
#     # Type: string
#     # Default: (randomly generated)
#     # Required: no
#     password: passwd
```
